# Field Guide — content

Public delivery of the Field Guide lesson corpus. `lessons.json` is the built array the
iOS app fetches at launch (with the bundled copy as offline fallback). It is generated by
the pipeline in the private `fieldguide` repo (`content/build-lessons.mjs`) — do not edit
by hand here.

Served via GitHub Pages:
`https://bladetho.github.io/fieldguide-content/lessons.json`
